# OpenHearth — Designs

This directory holds all **design work** for OpenHearth, per repo convention.

Use it for:

- **Wireframes & mockups** of the 10-foot UI (home screen, service grid,
  library browse, player, settings).
- **Focus/navigation maps** — how directional (D-pad/arrow) navigation moves
  between elements.
- **Visual language** — type scale, color/contrast for TV viewing, spacing,
  safe-area margins.
- **Component specs** — tiles, rows, focus indicators, player controls.
- **Flow diagrams** — launch-and-return (Home/Back), playback decision flow,
  config-driven layout.

## Conventions

- Source design files (Figma exports, Excalidraw, SVG, etc.) and rendered
  images both live here.
- Keep a short `.md` index alongside binary assets describing what each file
  is and which PRD requirement it supports.
- Design work supports the requirements in [`../docs/prd.md`](../docs/prd.md);
  reference the relevant requirement IDs (e.g. `FR-A1`, `§14.3`) where helpful.
